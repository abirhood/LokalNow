LokalNow 📍
One-time free setup — takes 5 minutes
Why this step? LokalNow needs a free Firebase database so all your friends can share posts, likes and comments in real time. Firebase is 100% free for small groups.
1
Go to console.firebase.google.com and sign in with Google
2
Click "Add project" → name it lokalnow → skip Analytics → Create project
3
Click the </> Web icon → Register app as lokalnow-web → copy the firebaseConfig values shown
4
In Firebase: Build → Firestore Database → Create database → Start in Test Mode (choose any region)
5
Paste your config values below and tap Save & Launch 🚀

— OR —
Skip for now — run in demo mode (posts not shared with friends)
🔒 Your Firebase config is saved in your browser only. It is never sent anywhere else.
💡 Firebase free tier: 1GB storage, 50K reads/day — plenty for a friend group.
